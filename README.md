# instab

Instab is an Shell Script to perform multi-threaded brute force attack against Instagram


# Installation

git clone https://github.com/Khalidhusain786/instab.git

cd instab

chmod +x instabshell.sh

chmod +x install.sh

service tor start

./install.sh

./instabshell.sh
